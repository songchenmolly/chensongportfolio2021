Overview

BoxFolio is a totally responsive slider plugin based on jQuery, HTML5 and CSS3. Looks great on any device. It can be bind to simple links or image elements on websites and it is suitable for portfolio, project showcase or even just for sliding simple transformed images. The special transformed images, portfolio items can be easily created with the Photoshop template included in the package. The following folder and file structure is coming with the plugin:

  • assets contains the psd mockup for creating the transformed images
  • boxfolio plugin folder
    • css folder of stylesheets
    • fonts folder of icon fonts
    • images folder of images (thumbnails and big ones which are presented by the plugin)
    • js folder of scripts
  • documentation

Please NOTE

The plugin is not compatible with Wordpress. BoxFolio is not a Wordpress plugin.

Installation

Setup style and scripts

Include the plugin's css files into head section and scripts into the bottom of the page before closing the body tag.

<!DOCTYPE html> 
<html lang="en">
<head>
    ...
    /* ======= BoxFolio style ======= */
    <link type="text/css" rel="stylesheet" href="boxfolio/css/boxfolio.core.css"> 
    <link type="text/css" rel="stylesheet" href="boxfolio/css/boxfolio.style.css"> 
</head> 
<body>
    ...
    /* ======= BoxFolio scripts ======= */
    <script type="text/javascript" src="boxfolio/js/boxfolio.core.js"></script>
    <script type="text/javascript" src="boxfolio/js/boxfolio.scripts.js"></script>
</body>
</html>
Bind plugin to a link

The plugin is implemented to watch the element's "boxfolio" ID. See the middle area below.

<!DOCTYPE html> 
<html lang="en">
<head>
    ...
    /* ======= BoxFolio style ======= */
    <link type="text/css" rel="stylesheet" href="boxfolio/css/boxfolio.core.css"> 
    <link type="text/css" rel="stylesheet" href="boxfolio/css/boxfolio.style.css"> 
</head> 
<body>
    ...
    ...
    /* ======= Simple link which will open the slider ======= */
    <a href="javascript:;" id="boxfolio" class="your-class__name">Your link's text </a>    
    ...
    ...
    /* ======= BoxFolio scripts ======= */
    <script type="text/javascript" src="boxfolio/js/boxfolio.core.js"></script>
    <script type="text/javascript" src="boxfolio/js/boxfolio.scripts.js"></script>
</body>
</html>

Extract item from psd mockup

Psd mockup can be found in assets folder. The following tutorial shows how to produce transformed images like in the demo.


If you have smaller image than the placeholder of the psd, follow the instructions below.

  • Open your image in Photoshop and put it into the placeholder of the boxfolio-item-mockup.psd
  • Check "Show Transform Controls"
  • Stretch the image right until the edges.

  • Accept
  • Uncheck the eye icon at background because it is not needed
  • File Save

  • Go back to the initial boxfolio-item-mockup.psd tab in Photoshop
  • Uncheck "Show Transform Controls"
  • Now we are ready to export the image.

  • File > Export > Save for web
  • Preferred dimension of the portoflio item image is 1000px X 720px.
  • Preferred dimension of the thumbnail image is: 300px X 150px.
